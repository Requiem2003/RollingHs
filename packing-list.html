Step 1: Get your Firebase config.
1. Go to console.firebase.google.com → Add project
2. Skip Google Analytics (not needed)
3. Click the web icon </> → register an app
4. Copy the firebaseConfig object
5. In the left sidebar: Build → Realtime Database → Create database → start in test mode
6. Paste your config below.
